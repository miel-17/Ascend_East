# HackFest2026-Salmons
Clarity - Your AI-powered research assistant and argument validator.

## 2. Clone the Repository
Open your terminal and clone the project:

```bash
git clone https://github.com/mielmandanas/HackFest2026-Salmons.git
cd HackFest2026-Salmons
```

## 3. Install Dependencies
Download all the necessary packages for the frontend to work:

```bash
npm install
```

## 4. Setup Environment Variables
Since Clarity leans on the Gemini API to intelligently parse research, you will need your API credentials.

1. Create a file named `.env.local` inside the directory.
2. Add your Gemini API key inside the file (e.g., `GEMINI_API_KEY=your_key_here`).

## 5. Start the Development Server
```bash
npm run dev
```

## 6. View the App
Open your web browser and go to **[http://localhost:3000](http://localhost:3000)** to see Proofline running!
